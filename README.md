# install
```bash
source install.sh
```

# commands

### new branch(nb)
```
git nb {機能名}. 
```  
これを入れることで. 
git checkout -b "{修正タイプ}/{gitのユーザー名}/{機能名}". 
を実行してくれるコマンド.
